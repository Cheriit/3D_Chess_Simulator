# 3D Chess simulator
This project is a simple C++ OpenGL chess gameplay simulator. Presented game is played based on gameplay file `game.txt`. 
Available moves:

Move: 
```
M <from> <to>
```

Castling: 
```
C <from_1> <to_1> <from_2> <to_2>
```

Promotion:

```
P <place> 2
```

## Compiling and running project (Linux)
```
run.sh
```
